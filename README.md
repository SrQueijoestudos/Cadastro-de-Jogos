# 🎮 Cadastro de Jogos

Um sistema simples de **cadastro de jogos** desenvolvido em **Portugol**, com foco em praticar vetores, estruturas condicionais, menus e interação com o usuário.

## 📋 Funcionalidades

- 📌 Menu interativo
- 🔍 Pesquisa de jogos cadastrados
- ⭐ Avaliação de jogos (nota de 0 a 10)
- 🚪 Opção para encerrar o programa

## 🛠️ Tecnologias utilizadas

- Portugol
- Portugol Studio / Portugol.dev

## 📚 Conceitos praticados

- Vetores
- Estruturas de decisão (`se`, `senao`, `escolha`)
- Estruturas de repetição (`enquanto`)
- Entrada e saída de dados (`leia` e `escreva`)
- Variáveis de diferentes tipos (`inteiro`, `cadeia`, `real` e `logico`)

## ▶️ Como executar

1. Abra o arquivo `Cadastro de Jogos.por` no Portugol Studio ou Portugol.dev.
2. Execute o programa.
3. Utilize o menu para navegar pelas opções.

## 📂 Estrutura do projeto

```
Cadastro de Jogos.por
README.md
```

## ⚠️ Observações

Este projeto foi desenvolvido para fins de aprendizado. Algumas melhorias podem ser implementadas futuramente, como:

- Cadastro completo de jogos (o menu possui a opção, mas ainda pode ser expandida).
- Remover a repetição de vários `se/senao` utilizando laços de repetição.
- Exibir a lista completa de jogos cadastrados.
- Editar e excluir jogos.
- Mostrar a média das avaliações.
